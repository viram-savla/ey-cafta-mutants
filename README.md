# BAML Risk Command Center

A production-grade treasury risk management tool for Bharat Advanced Materials Limited (BAML) — built for the EY CAFTA Case Championship 2026, 11th Edition.

## What It Does

- **Real-time scenario engine**: Drag sliders for iron ore shock, INR rate, and freight — see EBITDA margin and Board floor compliance update instantly
- **10 KPI Board Dashboard**: Live RAG-coded KPIs linked to the financial model
- **Nigeria Liquidity Tracker**: 12-month buffer forecast with facility simulation
- **Monte Carlo CFaR Simulator**: 1,000-path EBITDA margin distribution with 5th percentile CFaR

## Built With

React 18 + Vite | Recharts | Framer Motion | Tailwind CSS v4

## Case Context

BAML faces a 260bps EBITDA margin compression from unhedged iron ore, coking coal, freight and FX exposures. This dashboard implements the treasury risk framework from our EY CAFTA submission — transforming the Excel model into a real CFO tool.

Model: 10 Excel sheets | 7,382 formulas | Zero errors | Monte Carlo: 1,000 paths | SOFR: 3.59% (NY Fed, May 19 2026)
